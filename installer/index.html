How to set up the offline‚ö°Ô∏èSwitch?


The setup is relatively simple. It is done in three steps.


  1. 1. Flash the firmware on the ESP32
  2. 2. Create and set up a LNbits wallet
  3. 3. Configure the offline LN Switch via the touch display

First note: There are two different versions of the Sunton 3.5" 320*480 Smart Display (ESP32-3248S035). Once with ‚ÄúResistive Touch Screen‚Äù (pressure sensitive, foil like screen coating) and then with a ‚ÄúCapacitive Touch Screen‚Äù (glass display, reacts to conductivity). You can feel the difference, or see it on the back. See the pictures in the attachment. I would recommend the capacitive version. It is a bit more expensive, but also less sensitive. When selecting the firmware version, look for the ‚Äúr‚Äù for resistive or ‚Äúc‚Äù for capacitive after the version number.
